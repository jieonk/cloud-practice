cURL Lab

아래 엔드포인트로 마음껏 연습하세요. IP당 전역 분당 120회, /limited는 분당 20회로 제한됩니다.

OpenAPI 스펙

1) 에코

curl -i "https://cloud-practice.onrender.com/echo?role=student" -H "X-Lab: curl" -d "hello=world"

2) 기본 인증

기본 계정: admin/secret (환경변수로 변경 가능)

curl -i -u admin:secret https://cloud-practice.onrender.com/auth

3) 파일 업로드

curl -i -F "file=@./README.md" https://cloud-practice.onrender.com/upload

4) 지연/리다이렉트/상태

curl -i https://cloud-practice.onrender.com/delay/1500
curl -iL https://cloud-practice.onrender.com/redirect
curl -i https://cloud-practice.onrender.com/status/418

5) 429 제한

for i in {1..25}; do curl -s -o /dev/null -w "%{http_code}\n" https://cloud-practice.onrender.com/limited; done

6) gzip / 페이지네이션

curl -i --compressed https://cloud-practice.onrender.com/gzip
curl -s "https://cloud-practice.onrender.com/items?page=3&size=5"

1) 필수 쿠키 실습 (동의 쿠키)

lab_consent=true 쿠키가 없으면 403, 있으면 200을 반환합니다.

1-1) 쿠키 없이 접근 → 403

curl -i https://cloud-practice.onrender.com/cookie/require

1-2) 쿠키 설정 후 접근 → 200

# 서버에서 쿠키 발급
curl -i "https://cloud-practice.onrender.com/cookie/set?name=lab_consent&value=true"
# 같은 세션 유지하며 접근 (브라우저는 자동 전송, cURL은 -b/-c 옵션 사용)
curl -i -b "lab_consent=true" https://cloud-practice.onrender.com/cookie/require

1-3) 쿠키 저장/재사용 (cookie jar)

# 쿠키 파일에 저장하고 재사용
curl -c jar.txt "https://cloud-practice.onrender.com/cookie/set?name=lab_consent&value=true"
curl -b jar.txt https://cloud-practice.onrender.com/cookie/require

1-4) 쿠키 삭제

curl -i "https://cloud-practice.onrender.com/cookie/clear?name=lab_consent"
curl -i https://cloud-practice.onrender.com/cookie/require  # 다시 403

2) 파일 업로드 고급

단일/다중 업로드, 이미지 MIME 제한을 실습합니다.

2-1) 단일 업로드 (기존)

curl -i -F "file=@./README.md" https://cloud-practice.onrender.com/upload

2-2) 여러 파일 업로드 (최대 3개)

curl -i -F "files=@./a.txt" -F "files=@./b.txt" https://cloud-practice.onrender.com/uploads

2-3) 이미지 전용 업로드 (이미지만 허용)

curl -i -F "image=@./photo.png" https://cloud-practice.onrender.com/upload-image
# 이미지가 아니면 400 + "Only image/* allowed"

3) 쿠키 + 인증 조합

동의 쿠키가 있는 상태에서 Basic 인증까지 요구하는 흐름은 cURL 옵션 조합 연습에 좋습니다.

# 동의 쿠키 저장 -> 인증 접근
curl -c jar.txt "https://cloud-practice.onrender.com/cookie/set?name=lab_consent&value=true"
curl -i -b jar.txt -u admin:secret https://cloud-practice.onrender.com/auth

4) 트러블슈팅 팁

# 상세한 요청/응답 보기
curl -v https://cloud-practice.onrender.com/echo

# 헤더만 보기
curl -I https://cloud-practice.onrender.com/status/418

# 응답 본문을 파일로 저장
curl -o out.json https://cloud-practice.onrender.com/echo?role=student
