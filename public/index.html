cURL Lab

아래 엔드포인트로 마음껏 연습하세요. IP당 전역 분당 120회, /limited는 분당 20회로 제한됩니다.

OpenAPI 스펙

1) 에코

curl -i "https://cloud-practice.onrender.com/echo?role=student" -H "X-Lab: curl" -d "hello=world"

2) 기본 인증

기본 계정: admin/secret (환경변수로 변경 가능)

curl -i -u admin:secret https://cloud-practice.onrender.com/auth

3) 파일 업로드

curl -i -F "file=@./README.md" https://cloud-practice.onrender.com/upload

4) 지연/리다이렉트/상태

curl -i https://cloud-practice.onrender.com/delay/1500
curl -iL https://cloud-practice.onrender.com/redirect
curl -i https://cloud-practice.onrender.com/status/418

5) 429 제한

for i in {1..25}; do curl -s -o /dev/null -w "%{http_code}\n" https://cloud-practice.onrender.com/limited; done

6) gzip / 페이지네이션

curl -i --compressed https://cloud-practice.onrender.com/gzip
curl -s "https://cloud-practice.onrender.com/items?page=3&size=5"
